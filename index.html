Introducing Libem - Your Entity Solution

Libem is an innovative, open-source AI toolchain that transforms entity resolution using large language models (LLMs). It streamlines entity matching (EM)—the task of identifying if two descriptions refer to the same entity—crucial for data integration. Unlike traditional rule-based or rigid LLM-based systems, Libem uses a modular, adaptive approach for dynamic tool use, self-refinement, and optimization. It offers a reusable, composable toolchain that integrates into applications or APIs, configurable to use various models like GPT-4o, GPT-3.5-turbo, and Llama3, and tools for browsing and data preparation.
